# 기능 목록



## 클래스

1. Main

2. Game

3. Baseball
4. IOController



## 메소드

- Main.main() 
  - baseball 게임을 실행한다.
  - 게임을 재시작한다.
  - 게임을 종료한다.
- Game.start()
  - 게임을 시작한다.
- Game.play() 
  - 숫자를 입력받는다.
  - 결과를 출력한다.
- Game.selectNextAction()
  - 게임 재시작 여부를 묻는다.
  - 사용자 입력을 받는다.
- Baseball.generateAnswer() 
  - 컴퓨터의 숫자를 생성한다.
- Baseball.matchNumber() 
  - 입력한 숫자와 정답을 비교한다.
- IOController.getNumberOfUser() 
  - 사용자의 숫자를 입력받는다.
- IOController.printResult()
  - match 결과를 출력한다.
